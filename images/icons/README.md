# Iconos Personalizados para Dashboard

Esta carpeta contiene los iconos personalizados usados en las tarjetas estadísticas del dashboard de estudiantes.

## Iconos Requeridos

Coloca tus imágenes en esta carpeta con los siguientes nombres:

### 1. `teacher.png`
- **Uso:** Card "Lecciones Completadas" (morada)
- **Descripción:** Icono de profesor/enseñanza
- **Tamaño recomendado:** 64x64px o 128x128px
- **Formato:** PNG con fondo transparente

### 2. `gamepad.png`
- **Uso:** Card "Juegos Jugados" (cyan)
- **Descripción:** Icono de control de videojuegos/gamepad
- **Tamaño recomendado:** 64x64px o 128x128px
- **Formato:** PNG con fondo transparente

### 3. `trophy.png`
- **Uso:** Card "Marcos Desbloqueados" (naranja)
- **Descripción:** Icono de trofeo/medalla/logro
- **Tamaño recomendado:** 64x64px o 128x128px
- **Formato:** PNG con fondo transparente

### 4. `timer.png`
- **Uso:** Card "Tiempo Total" (verde)
- **Descripción:** Icono de cronómetro/reloj
- **Tamaño recomendado:** 64x64px o 128x128px
- **Formato:** PNG con fondo transparente

## Notas Técnicas

- **Formato recomendado:** PNG con transparencia
- **Tamaño óptimo:** 64x64px o 128x128px (para pantallas retina)
- **Peso máximo:** 50KB por archivo
- **Color:** Los iconos se verán sobre fondos de colores (morado, cyan, naranja, verde), así que asegúrate de que sean visibles

## Alternativas

Si prefieres usar SVG en lugar de PNG:
- Usa los mismos nombres pero con extensión `.svg`
- Actualiza las referencias en `app/dashboard.html`
- Los SVG son escalables y tienen mejor calidad
